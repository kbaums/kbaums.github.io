2018 Site Redesign Mockups - Made in Sketch

Home

The home page will feature a mouse interaction using Pixi.js. Users will be able to make ripples on the background which will look like it is underwater.

Work

The work page will show my seven most recent projects in a vertical carousel, which will not cycle automactically. There will be a list of the skills I used for each project and a short description of the project.

About

The about page will feature a short autobiography and a fun interaction with the photo. By moving the mouse over my photo, it will unmask a photo of my dog. This will be done using Pixi.js.

Header Interaction

By scrolling over the social link in the navbar, "social" will turn into three different social links.
